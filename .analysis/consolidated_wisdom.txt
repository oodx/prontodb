🔍 KEY ARCHITECTURAL PATTERNS
================================================================================
1. RSB Framework Integration
   - String-biased APIs
   - Standard entry patterns (args!, pre_dispatch!)
   - Lifecycle command support

2. Storage Architecture
   - SQLite backend with WAL mode
   - Multi-database support via cursors
   - User and context isolation
   - XDG-compliant storage paths

3. Multi-Agent Features
   - User-level isolation
   - Cursor-based context switching
   - TTL cache namespaces
   - Meta namespace support
